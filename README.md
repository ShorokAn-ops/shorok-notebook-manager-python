# Personal Notebook Manager

This project is a simple Python command-line application for managing personal notes.

## How to Run

1. Download or clone this repository.
2. Make sure Python 3 is installed on your computer.
3. Open a terminal in the project directory.
4. Run the program using:
   python shorok_Personal_Notebook_Manager.py
5. Follow the on-screen menu instructions.

## Notes

- The program runs entirely in the terminal.
- Notes are stored locally in a file called `notes.json`.
